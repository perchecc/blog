# 注册服务号

## 概念

> 服务号是微信提供给企业或者个体工商户的一种公众号类型，所以我们注册服务号，必须要有工商营业执照

## 准备

- 工商营业执照
- 注册服务号的一个邮箱
- 公众号管理员信息
  - 姓名
  - 身份证号
  - 能接收验证码的手机号
  - 一个微信号（必须绑定了银行卡且绑定了发送验证码的手机号的）
